# Predictive Modeling Using Machine Learning

## 🤖 1. Supervised Learning Model
* **Algorithm Used:** Random Forest Classifier
* **Target Feature:** `Survived` (Binary Classification: 0 = No, 1 = Yes)
* **Dataset Split:** 80% Training Data, 20% Testing Data

## 📈 2. Model Performance Evaluation
* **Overall Test Accuracy:** ~81.5% (The model correctly predicts passenger survival status over 81% of the time).
* **Precision & Recall:** The model exhibits balanced metrics across both classes, maintaining high predictive reliability.
